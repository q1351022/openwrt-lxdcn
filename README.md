# OpenWrt LXD 一键部署脚本

本项目提供一个自动化脚本，用于在 Linux 系统上通过 LXD 容器部署 OpenWrt。

## 功能特性

- 支持主路由 / 旁路由部署模式
- 支持桥接自动创建
- 支持自动检测架构下载对应 OpenWrt 镜像
- 支持 IPv6、Wi-Fi 热点切换（可配置开关）
- 支持容器快捷命令别名
- 支持网络多环境（有线、USB、无线）自适应
- 支持 OpenWrt 最新版本下载或默认版本安装

## 快速开始

```bash
bash openwrt-lxd-setup.sh
```

首次运行请选择“一键部署”，按提示完成安装。

## 高级功能开关

运行后可从菜单中开启或关闭以下功能：

- IPv6 支持
- Wi-Fi 热点自动切换支持
